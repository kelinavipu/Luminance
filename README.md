# 🌬️ Luminance: Your Digital Wellbeing Sanctuary

**Luminance** is not just another screen time tracker. It is a high-fidelity, AI-powered digital sanctuary designed to help you reclaim your focus and master your digital habits. Using predictive behavioral analytics and a touch of professional sarcasm, Luminance guides you toward a more intentional life.

---

## ✨ Key Features

### 🧘‍♂️ Zen Blocker & Iron Lock
When you set a limit, we mean it. The **Iron Lock Blocker** uses a full-screen, opaque **Blooming Lotus** overlay to ground you in the present moment, preventing mindless scrolling with a "Decision Tree" backed logic.

### 🧠 Predictive Behavioral Analytics
Powered by custom regression models, Luminance analyzes your last 7 days of usage to suggest **"Golden Thresholds"**. 
*   **Social/Games:** Heavy reduction suggestions to break the habit loop.
*   **Incremental Change:** For heavy users (>3 hrs/day), the AI suggests "Little by Little" transitions (15% vs 30% cuts) to ensure sustainable progress.

### 📊 The Analytical Suite (6-Slider Dashboard)
A premium, multi-layered dashboard providing total transparency:
1.  **Usage Sanctuary:** Real-time tracking with the Breathing Lotus aura.
2.  **Mastery Streaks:** Track your "Glow Days" (100% completion) and Victories.
3.  **Risk Assessment Audit:** A granular breakdown of what contributes to your digital guilt (Night usage, pickups, wasteful binging).
4.  **Weekly Benchmarks:** Macro-averages to track your progress over time.
5.  **Usage vs. Guilt Trends:** Visualizing the correlation between app time and wellbeing.
6.  **Daily App Audit:** A complete footprint of every app opened today, highlighting your "Top Habits".

### 🤖 Sarcasm Engine & AI Guidance
Integrated with **Groq AI (Llama 3)**, Luminance provides daily coaching. Sometimes it's a gentle nudge; sometimes it's a sarcastic reality check about your 4-hour YouTube spiral.

*   **Frontend:** Flutter (Dart)
*   **Backend:** Firebase (Firestore, Auth)

*   **Native Integration:** Android UsageStatsManager & AccessibilityService
*   **State Management:** Provider

## 🛠️ Setup & Installation

### Prerequisites
*   Flutter SDK (Stable)
*   Android Studio / VS Code
*   A Firebase Project (GoogleServices.json required)

### Steps
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/luminance.git
    ```
2.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Native Permissions:**
    *   Enable **Usage Access** in Android Settings.
    *   Enable **Accessibility Service** for Luminance.
    *   Set **Battery Optimization** to "No Restrictions" (Critical for background blocking).
4.  **Run the App:**
    ```bash
    flutter run
    ```
